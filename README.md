# 媒体播放器



## 开发环境：
- C++17
- Qt5.14.2 MinGW 64-bit

## 功能介绍：

* 跳转播放：前进、后退
* 暂停
* 音量调节
* 拉动进度条
* 视频画面固定比例缩放
