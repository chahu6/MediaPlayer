# 媒体播放器

## 项目描述：

媒体播放器，QT版本：QT5.14.2，使用了SDL2-2.26.5、ffmpeg-64bit和eigen-3.4.0

## 开发环境：
- C++17
- Qt5.14.2 MinGW 64-bit

## 功能介绍：

* 跳转播放：前进、后退
* 暂停
* 音量调节
* 拉动进度条
* 视频画面固定比例缩放
